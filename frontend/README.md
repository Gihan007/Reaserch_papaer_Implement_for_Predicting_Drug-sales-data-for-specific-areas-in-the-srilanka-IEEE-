# Drug Sales Prediction Frontend

This is a modern ReactJS frontend with Tailwind CSS for the Drug Sales Prediction Dashboard.

## Features
- Sleek UI with gradient backgrounds and smooth animations
- Select drug category, prediction model, and weeks ahead
- Displays forecast value and interactive chart
- Responsive design for all devices
- Uses Axios for API calls, Chart.js for visualizations, and React Router for potential future navigation

## Setup
1. Install dependencies:
   ```
   cd frontend
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

3. Open `http://localhost:3000` in your browser.

## Backend
Ensure the Flask backend is running on `http://127.0.0.1:5000` for API calls.

## Tech Stack
- ReactJS
- Tailwind CSS
- Axios
- Chart.js
- React Router DOM
